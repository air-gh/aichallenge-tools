## patch自動評価サーバ

#### 使い方

パッチ形式のファイルを本ディレクトリに置く

```
ex)
# 以下で取得した*.patch形式のファイルを本ディレクトリに置き、1h程待つ。(左上の+add/upload filesを使えばOK)
cd ~/aichallenge2023-sim
git diff > 20230815.patch
```
